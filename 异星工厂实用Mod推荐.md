# 异星工厂实用 Mod 全分类推荐

> 整理时间：2026年6月  
> 兼容版本：Factorio 2.0 + Space Age DLC  
> 数据来源：[Factorio 官方 Mod 站](https://mods.factorio.com/)

---

## 🧮 规划与计算

| Mod | 下载量 | 用途 |
|-----|--------|------|
| **Helmod** | 52.6 万 | 游戏内产线计算器，算机器数、原料、电力。支持插件、信标、多人协作。快捷键：`U` 打开主面板，`O` 打开配方选择器 |
| **Factory Planner** | 31.5 万 | Helmod 的轻量替代品，UI 更直观，仍在活跃更新 |
| **Rate Calculator** | ~30 万 | 框选机器即时显示输入/输出速率，秒算配平 |
| **Recipe Book** | 22.7 万 | 配方百科全书——这东西怎么造？这东西有什么用？ |
| **FNEI** | 经典 | Recipe Book 的前辈，同样好用 |

## 🔍 信息与监控

| Mod | 下载量 | 用途 |
|-----|--------|------|
| **Bottleneck Lite** | ~40 万 | 机器状态指示灯 🟢🟡🔴，零性能消耗（无运行时脚本）。绿色=正常，红色=缺料/停电/停工，黄色=输出堵塞。改设置需重启游戏 |
| **Resource Monitor** | ~1 万 | 自动监控矿脉剩余量和预计耗尽时间，支持多星球。YARM 的现代替代品 |
| **YARM** | 16.6 万 | 经典矿脉监控工具，但已很少更新。作者建议迁移到 Resource Monitor |
| **Informatron** | 67.4 万 | 游戏内帮助浏览器，大型 Mod 的文档会存放于此。默认快捷键：`I` |
| **Milestones** | 经典 | 记录关键科技解锁时间点，有成就感 |

## 🏗 建造辅助

| Mod | 下载量 | 用途 |
|-----|--------|------|
| **Squeak Through 2** | 25.2 万 | **必装**。可以在管道、电线杆、太阳能板之间穿行。原版 Squeak Through 已停更，务必装带「2」的版本 |
| **Even Distribution** | ~25 万 | Ctrl+Shift+拖拽 → 物品均分到多个机器/箱子。省去疯狂右键的重复劳动 |
| **VehicleSnap** | ~20 万 | 开车自动对齐网格，16向对齐，告别歪歪扭扭的停车 |
| **Long Reach** | 经典 | 增加互动距离，不用走到面前才能操作 |
| **Nanobots 2** | 经典 | 研发机器人之前就能用的微型建造机器人，消耗弹药式纳米机器人，对平衡影响很小 |
| **Fill4Me** | ~15 万 | 放下熔炉/炮塔/车辆时自动装入燃料和弹药 |
| **Auto Deconstruct** | ~15 万 | 矿机/油井耗尽时自动标记拆除，配合机器人自动化清理 |

## ✅ 新手推荐：第一局装什么？

| 优先级 | Mod | 理由 |
|--------|-----|------|
| 🟢 **必装** | Squeak Through 2 | 不在管道间卡住，幸福感提升 100 倍 |
| 🟢 **必装** | Even Distribution | 省下无数按右键的时间 |
| 🟢 **强烈推荐** | Rate Calculator | 框选秒算产量，不用心算配平 |
| 🟡 **推荐** | VehicleSnap | 开车不再歪歪扭扭 |
| 🟡 **推荐** | Bottleneck Lite | 一眼看出哪台机器出问题 |
| 🟡 **推荐** | Helmod 或 Factory Planner | 产线不再靠猜，精确计算 |
| 🔵 **可选** | Recipe Book | 查配方不再翻科技树 |
| 🔵 **可选** | Lighted Electric Poles | 不需要满地图铺路灯 |

> 以上 8 个 Mod 全装也对游戏机制**零改动**，不影响游戏内成就追踪，不影响原版体验。唯一的"影响"是所有重复劳动被消除，体验大幅提升。

---

## ⚠️ 关于 Steam 成就

- 所有 Mod 都会导致 **Steam 成就同步被禁用**
- 但**游戏内成就追踪正常**（可以用 Mod 成就追踪器替代）
- 纯 UI/QoL Mod 不会改变游戏玩法
