## 各子合集详解

### [0] 双轨窄主干线（5 蓝图）

最常用的轨道制式，轨道间距紧凑（适合早期）：

| 蓝图 | 用途 |
|------|------|
| 直轨 | 无限铺的主干线 |
| 掉头弯 | U 形掉头 |
| 占位符 | 预留未来交叉口位置 |
| 弯头 | 90° 转弯 |
| 十字交叉 | 平面十字口 |

### [1] 双轨宽主干线（5 蓝图）

双轨但间距更宽（大交叉口更流畅），和大路口/四轨兼容更好。

### [2] 四轨主干线（10 蓝图）⭐ 核心

超高吞吐量——每条方向 2 条轨道，共 4 条：

| 蓝图 | 用途 |
|------|------|
| 直轨 | 四轨并行 |
| 掉头/U 形/弯头/全互通 | 各种交叉口 |
| **拥堵控制（Congestion Control）** | 通过信号灯和分叉控制车流，避免四轨口死锁 |
| 窄双轨/宽双轨连接 | 四轨→双轨的过渡口 |
| 四轨车站出口 | 车站出口汇入任意一条轨道 |

### [3] 车站轨道（12 蓝图）

把火车从主干线引出，进站卸装。支持：

```
单站 2 厢（双向）    单站 4 厢 + 2 候车位
                      单站 8 厢 + 停车场
双站 4 厢              三站 4 厢
四站 4 厢              四站 8 厢 + 8 候车位
```

### [4] 升级/拆除规划器（9 个）

预配置的升级规划器和拆除规划器，一键切换车站系统。

### [5] Provider 供应站（26 蓝图）⭐

**装货站。** 每种配置分三种调度系统 + 三种装货方式：

```
Header（必装）:
  - Vanilla Item Provide Header
  - LTN Item Provide Header
  - Cybersyn Item Provide Header

Body（按车厢数选）:
  1/2/4/8 厢 × 左输入/右输入/合并箱
  1/2/4/8 罐车
```

### [6] Requester 请求站（31 蓝图）⭐

**卸货站。** 同样分三种系统，额外支持 2 条蓝带快速输出：

```
Header + Body:
  1/2/4/8 厢 × 左输出/右输出/合并箱/2蓝带输出
  1/2/4/8 罐车
```

### [7] 支持站（6 蓝图）

总控和车库：

| 蓝图 | 用途 |
|------|------|
| Vanilla Depot Hub | 原版车库 |
| Vanilla Controller | 原版总控器 |
| Vanilla Refueling Station | 加油/加水站 |
| 8/12-train LTN Depot Hub | LTN 车库（煤炭火车供燃料） |
| Cybersyn Depot Hub | Cybersyn 车库 |

### [8] 原版火车（11 蓝图）

1/2/4/8 厢 + 液体罐车 + 双向车的预配置时刻表。

### [9] LTN 火车（10 蓝图）

同上，预配置了 LTN 调度参数。

### [10] Cybersyn 火车（11 蓝图）

同上 + 特殊任务火车（如补燃料、清垃圾）。

### [12] 附加工具（5 蓝图）

| 蓝图 | 用途 |
|------|------|
| **Multiplexer / Demultiplexer** | 多路信号复用/解复用——一根线传多路信号 |
| **Train Detector** | 检测轨上是否有火车 |
| **Train Rate Display** | 显示火车通过频率 |

### [13] 杂项（7 蓝图）

包括 4 厢/8 厢轨道长度标尺、四轨模板、小型/中型机器人城市区块。

## 怎么用

### 第一步：选轨道制式

```
小工厂 → 双轨窄 [0]
大工厂 → 双轨宽 [1] 或 四轨 [2]
```

### 第二步：选调度系统

```
原版中断 → [5]/[6] Vanilla Header
LTN      → [5]/[6] LTN Header
Cybersyn → [5]/[6] Cybersyn Header
```

> ⚠️ **同一个星球上只能用一种系统！** LTN 和 Cybersyn 互不兼容。

### 第三步：车站 = Header + Body

```
1. 先放 Header（Vanilla/LTN/Cybersyn 选一个）
2. 再放 Body（按车厢数和方向选）
3. 两个蓝图拼起来 = 完整车站
```

### 第四步：放车库

```
原版 → Vanilla Depot Hub
LTN  → 8/12-train LTN Depot Hub
Cybersyn → Cybersyn Depot Hub
```

### 第五步：放火车

```
从 [8]/[9]/[10] 中选对应车厢数的火车蓝图
放到铁轨上 → 自动开去车库注册
```
