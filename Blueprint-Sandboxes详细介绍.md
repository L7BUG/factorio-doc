# Blueprint Sandboxes 详细介绍

> 来源：[Factorio 官方 Mod 站](https://mods.factorio.com/mod/blueprint-sandboxes)  
> 作者：somethingtohide | 版本：3.2.2 | 下载量：~10.5 万  
> 兼容：Factorio 1.1 ~ 2.0 + Space Age DLC

---

## 是什么？

一个让你在**游戏内就能测试蓝图**的 Mod。你不需要退出当前存档、不需要开"沙盒模式"新图——按一个快捷键，直接切到一个独立的创作空间，设计、测试、调整蓝图，然后带着成品回到真实世界。

---

## 为什么需要它？

### 没有 Blueprint Sandboxes 时：

```
  游戏里想测试一个产线设计
       │
       ▼
  存档 → 退到主菜单 → 开新图（沙盒模式）
       │
       ▼
  编辑器里测试蓝图 → 发现问题 → 改 → 再测
       │
       ▼
  保存蓝图字符串 → 退到主菜单 → 加载原存档
       │
       ▼
  导入蓝图 → 发现忘了什么东西 → 重复以上流程 😫
```

### 有 Blueprint Sandboxes 时：

```
  按 Shift+B → 进入沙盒 → 测试蓝图 → Ctrl+C 复制
      │
      ▼
  按 Shift+B 回游戏 → Ctrl+V 粘贴 ✅  几秒钟搞定
```

## 核心功能

### 🎮 上帝模式

沙盒内你拥有完整编辑器能力：

| 能力 | 说明 |
|------|------|
| **飞行** | 自由飞行，无视地形和碰撞 |
| **即时建造/拆除** | 点击即完成，不需要机器人和等待时间 |
| **虚影自动建造** | 放下虚影（ghost）自动变为实体 |
| **无敌** | 敌人不存在，不会受伤 |

### ♾ 无限资源

| 工具 | 用途 |
|------|------|
| **无限箱子 (Infinity Chest)** | 生成无限任意物品，测试原料供应 |
| **无限装卸机 (Infinity Loader)** | 直接向机器注入/取出物品，测试吞吐量 |
| **无限管道** | 提供无限流体，测试化工产线 |

### 🌍 星球模拟

沙盒可以调整物理属性来模拟不同星球：

| 属性 | 用途 |
|------|------|
| **重力** | 模拟 Fulgora 低重力或 Aquilo 高重力 |
| **气压** | 影响流体行为 |
| **地面类型** | 可铺太空平台地面，在沙盒里设计飞船 |

### 📋 蓝图输入输出

| 方式 | 操作 |
|------|------|
| **复制粘贴** | 沙盒里 Ctrl+C → 回到游戏 Ctrl+V |
| **蓝图库** | 沙盒里保存到蓝图库，游戏里直接取出 |
| **光标转移** | 拿在光标上的蓝图可以直接带入/带出沙盒 |

### 🎒 背包隔离与恢复

| 阶段 | 状态 |
|------|------|
| **进入沙盒时** | 背包自动保存 |
| **沙盒内** | 拥有独立背包，随便放东西 |
| **退出沙盒时** | 背包恢复到进入前的状态（沙盒里拿的东西不会带回现实） |

## 两种隔离模式

2.0 版本新增了隔离级别选择，这是一个**非常重要的设计选择**：

### 🏝 完全隔离（Full Isolation，历史默认）

```
┌──────────────┐     ┌──────────────────┐
│  真实世界      │     │  沙盒世界          │
│              │     │                  │
│  你的角色     │ ╳  │  上帝控制器        │
│  你的背包     │ ╳  │  独立背包          │
│  你的科技     │ ╳  │  可选全科技解锁     │
│  你的势力     │ ╳  │  独立势力          │
│              │     │                  │
└──────────────┘     └──────────────────┘
       完全隔离，互不影响
```

| 优点 | 缺点 |
|------|------|
| 不可能意外作弊 | 你的真实角色留在原地，可能被虫子攻击 |
| 可以解锁全科技测试 | 无法远程查看沙盒 |
| 更安全 | 进出时 Undo 历史会丢失 |

### 🔗 无隔离（None Isolation，2.0 新增，推荐）

```
┌──────────────────────────────────────────┐
│              真实世界                      │
│                                          │
│  你的角色站在原地（安全！）                  │
│       │                                  │
│       │  远程视图 (Remote View)            │
│       ▼                                  │
│  ┌──────────────────┐                    │
│  │  沙盒世界          │                    │
│  │  (通过地图远程操作)  │                    │
│  │                  │                    │
│  │  - 科技同步       │                    │
│  │  - 警告同步       │                    │
│  │  - 物流组同步     │                    │
│  └──────────────────┘                    │
│                                          │
└──────────────────────────────────────────┘
     你从未离开你的角色，只是远程操作沙盒
```

| 优点 | 缺点 |
|------|------|
| 角色永远不会受到威胁 | 不能用"全科技"模式 |
| 科技/物流/警告实时同步 | 沙盒内成就会触发 |
| 始终可以访问 | — |
| Undo/Redo 正常工作 | — |

> **推荐**：除非有特殊原因，**用「无隔离」模式**——更安全、更简单、更融入游戏。

## Space Age 特殊用法

### 🚀 在沙盒里设计太空平台

沙盒**不是一个真正的星球**——它不能放置真正的太空平台中枢（Hub），但可以用替代方案：

| 步骤 | 操作 |
|------|------|
| 1 | 用瓷砖选择器选「太空平台地面」，铺出一大片 |
| 2 | 用**货物着陆台（Cargo Landing Pad）**代替平台中枢（尺寸相同，功能相近） |
| 3 | 正常放置推进器、采集器、炮塔等太空建筑 |
| 4 | 在沙盒设置 → 电力页 → 开启「自动配电」（太空平台不需要电线杆） |
| 5 | 设计完成后复制蓝图，带回游戏粘贴到真实平台上 |

## ⚠️ 已知限制

| 限制 | 说明 |
|------|------|
| 🌊 **不能模拟星球特效** | Aquilo 的冰冻、Fulgora 的闪电、Gleba 的腐化时间——沙盒做不到，这是引擎限制 |
| 🎵 **音乐消失** | 进沙盒后背景音乐停止。修复：按住 Ctrl+Alt 点设置 → Mod 设置 → 开启 `ambient-music-based-on-physical-location` |
| 🏗 **不能放太空平台中枢** | 会创建真实平台而非模拟 |
| 📜 **Undo 问题** | 完全隔离模式下进出沙盒后 Undo 历史丢失（无隔离模式正常） |
| 🔬 **实验室** | 沙盒内不能放置实验室（设计限制） |
| 🏆 **懒人成就** | 沙盒内手搓物品会计入手搓统计，影响「Lazy Bastard」成就追踪 |

## 与 Editor Extensions 的对比

| | Blueprint Sandboxes | Editor Extensions |
|------|------|------|
| **定位** | 隔离的测试空间 | 在真实世界开启编辑器 |
| **安全性** | ✅ 不会改真实世界 | ⚠️ 可能在真实世界误操作 |
| **快捷切换** | Shift+B 秒切 | Ctrl+Alt+E 进编辑器 |
| **隔离模式** | 两种可选 | 无隔离概念 |
| **2.0 支持** | ✅ 新增远程视图模式 | 传统编辑器模式 |
| **性能** | 沙盒独立运行，不拖累主世界 | 编辑器在真实世界操作 |

> 两者**不兼容**（同时装会禁用 Editor Extensions 的 Lab 模式），选一个即可。

## 实际使用流程示例

```
你想设计一个每分钟产 100 个蓝瓶的化工厂：

  1. 在游戏里按 Shift+B → 进入沙盒

  2. 沙盒设置中选「无隔离」模式（科技同步）

  3. 放无限箱子提供铁板、铜板、塑料、硫磺

  4. 摆组装机 → 接管道 → 放化工厂 → 调配方

  5. 用量速计算器（Rate Calculator）验证产能

  6. 发现瓶颈 → 调整 → 再验证

  7. 满意后 Ctrl+C 框选整个产线

  8. 按 Shift+B → 回到游戏

  9. Ctrl+V 放下蓝图实体

  10. 让建筑机器人开工 ✅
```

## 一句话总结

> **Blueprint Sandboxes = 游戏内置的蓝图测验室。** 随时随地按 Shift+B 进入独立空间，用无限资源测试任何设计，验证通过后带回蓝图正式施工。和切出去开新图测试的体验相比，是天壤之别。Space Age 时代更是可以远程查看、安全测试太空平台设计。
