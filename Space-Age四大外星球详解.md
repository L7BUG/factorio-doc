## 1. 🌋 Vulcanus — 火山星球

### 为什么第一个去

铸造厂和大采矿机对后续所有产线是质的飞跃。基建舒服，太阳能无限。

### 独家建筑

| 建筑 | 作用 |
|------|------|
| **铸造厂** | 4 槽，自带 50% 产能，速度 4。熔岩→金属，跳过铁板中间步骤 |
| **大型采矿钻机** | 13×13 范围，资源消耗 -50%，矿脉寿命翻倍 |
| **涡轮传送带（绿带）** | 60件/秒，地下 10 格 |

### 资源体系

```
熔岩（免费） + 方解石 → 铸造厂 → 熔融铁/铜 → 铁板/铜板/钢/齿轮
煤（地表挖）→ 液化 → 重油 → 裂解 → 轻油/石油气 → 塑料
硫酸（火山喷泉）→ 酸中和发电 + 化工
钨（熔岩盆地，大采矿机采）→ 绿带 + 冶金科技包
```

### 电力

- 前期：太阳能 400% 输出
- 中后期：酸中和 → 500°C 蒸汽 → 汽轮机

### 敌人：拆迁虫

- Boss 型，各有固定领地
- 激光/火焰/爆炸**全免疫**——只用物理伤害
- 打死不复活，领地永久归你

---

## 2. ⚡ Fulgora — 雷暴废土

### 为什么第二个去

电磁工厂 5 槽 + 50% 产能 = 品质农场核心。机械装甲解锁飞行。

### 独家建筑

| 建筑 | 作用 |
|------|------|
| **回收机** | 逆转配方返还原料，品质循环的核心 |
| **电磁工厂** | 5 槽，自带 50% 产能，电路/插件的终极生产建筑 |
| **机械装甲** | 步行终极装备，自带飞行 + 多槽位 |
| **特斯拉炮塔** | 连锁闪电 AOE，Gleba 五足虫克星 |

### 资源体系

```
废料 → 回收机 → 12 种产物混合输出：

  铁齿轮 → 铁板    蓝电路 → 保留/拆解
  钢板             钬矿石 → 化工链（Fulgora 独占）
  红电路            固体燃料 → 烧/留
  低密度结构         冰 → 水
  电池 → 铜板       石头
  铜线 → 铜板       混凝土 → 回收做铁矿

二次回收：堵带子就销毁——回收机吃一切
```

### 电力

- 避雷针 → 夜间雷暴发电 → 蓄电池存电
- 每个岛屿雷击面积有限 → 发电上限被岛屿大小限死

### 敌人：无

---

## 3. 🍄 Gleba — 农业星球

### 为什么第三个去

生物实验室科研效率翻倍。蜘蛛机甲。但**最难的星球**——腐化机制颠覆所有工厂习惯。

### 独家建筑

| 建筑 | 作用 |
|------|------|
| **生化室** | 4 槽，靠营养物质驱动 |
| **农业塔** | 自动种植 + 收割 Yumako/Jellynut |
| **焚烧塔** | 烧有机物发电，250% 燃烧效率 |
| **生物实验室** | 科研终极建筑，**科技包效率翻倍** |
| **蜘蛛机甲** | 步行终极载具，铁轨上随便走 |

### 资源体系

```
Yumako树（绿区）+ Jellystem树（红区）
         │
   果实 → 加工 + 营养物质
         │
   铁菌 + 铜菌（活的！）
         │
   腐化 → 铁矿 + 铜矿

石油气/塑料/硫磺/润滑油 → 全来自果实生化配方
```

### 核心机制：腐化（Spoilage）

- 所有有机物有保质期
- 营养物质 5 分钟烂掉
- 五足虫卵不腐化但**会在你工厂里孵化**
- 产线不能停 → 停了全部烂掉 → 重启极痛苦
- 多余物秒送焚烧塔烧掉

### 电力

- 太阳能 50% 不划算
- 焚烧塔烧腐化废料 + 多余有机物 → 热交换器 → 汽轮机

### 敌人：五足虫

- 被孢子（不是污染）吸引
- 践踏者 15000 血，80% 激光抗，但**火焰无抗**
- 卵在工厂里孵化 → 内部爆破

## 4. ❄ Aquilo — 冰冻海洋

### 为什么最后去

需要前三个星球的科技包才能解锁 Aquilo 科技。最终挑战。

### 独家建筑

| 建筑 | 作用 |
|------|------|
| **低温工厂** | **8 槽**，无天生产能但槽位最多 |
| **聚变反应堆** | 终极电源，无限能量 |
| **聚变发电机** | 聚变等离子体 → 电力 |
| **轨道炮 / 轨道炮塔** | Aquilo 以外打巨型小行星必须 |
| **低温科技包** | 终局科技 |
| **加热塔** | 250% 燃烧效率，供热保命 |

### 核心机制：冰冻

- Aquilo 是 -100°C 的氨海冰洋
- **所有建筑必须紧邻热管**，否则冻住停摆
- 热管必须 ≥ 30°C
- 热管不散热，只给连接的建筑供热
- 机器人功耗 ×5

### 资源体系

```
氨海 → 水泵 → 氨溶液 → 氨 + 冰
原油 → 油井
锂卤水 → 抽油机 → 锂 → 锂板
氟 → 来自氨溶液处理

氨 + 氟 → 氟酮（热）→ 低温工厂 → 氟酮（冷）→ 聚变电池

必须进口：铁、铜、煤、石头、方解石
```

### 建筑热消耗速查

| 建筑 | 热消耗 |
|------|------|
| 传送带 | 10 kW |
| 管道 | 1 kW |
| 组装机 | 100 kW |
| 低温工厂 | 100 kW |
| 机器港 | 300 kW |
| 火箭发射井 | 300 kW |
| 铸造厂 | 300 kW |
| 信标 | 400 kW |

### 电力

- 太阳能 **1%**——别想了
- 加热塔烧火箭燃料供热 → 热交换器 → 汽轮机
- 终极：聚变反应堆

### 敌人：无
