# 降低污染完全指南

> 适用版本：Factorio 2.0 + Space Age DLC

---

## 核心原则

降低污染的优先级不是「别产」，而是**从源头砍**。污染的真正麻烦不是「看着难看」，是污染云覆盖虫巢 → 虫子进化加速 + 进攻频率暴增。

---

## 污染来源占比（典型中后期工厂）

| 来源 | 占总污染 |
|------|------|
| 🔴 **矿机** | **~40%** |
| 🟠 **熔炉** | **~30%** |
| 🟡 **组装机/化工厂** | ~15% |
| 🟢 **锅炉（蒸汽发电）** | ~10% |
| ⚪ 其他（炼油厂、泵站等） | ~5% |

> **矿机 + 熔炉 = 70% 的污染。** 你其他方面再怎么省，这两个不解决就是杯水车薪。

---

## 第一刀：砍矿机（污染削减 ~32%）

### 能效模块 1 ×3

```
原版电矿机：10 污染/分钟

每台矿机上 3 个能效模块 1 → 电耗 -80% → 污染降 80%
10 → 2 污染/分钟

矿区几十台矿机 → 全部上能效模块 → 污染雪崩式下降
```

| 投入 | 效果 |
|------|------|
| 3 个能效 1/台矿机 | 电耗 -80%，污染 -80% |

成本极低，效果极大。**这应该是你解锁能效模块后做的第一件事。**

---

## 第二刀：砍熔炉（污染削减 ~25%）

```
石炉/钢炉：烧煤，污染直接产生在炉子里

换成电炉 + 能效模块 ×2：
  电炉本身：0 污染
  电力来源：太阳能/核电 → 0 污染
```

电炉搭能效模块，熔炉端的污染可以降到接近零。

---

## 第三刀：切清洁电力

| 电源 | 污染 | 备注 |
|------|------|------|
| 锅炉烧煤 | 30/台 | 主力污染源之一 |
| 核电 | 0 | 采铀矿产生微量污染，忽略不计 |
| 太阳能 | 0 | 配合蓄电池，完全零污染 |
| 蓄电池 | 0 | 不产污染，只存电（不改变污染值） |

越早切太阳能/核电，污染越低。megabase 最终都切 100% 清洁电力。

---

## 第四刀：插件全覆盖

### 优先级

| 建筑 | 用什么 | 效果 |
|------|------|------|
| 矿机 | 能效 1 ×3 | 电耗 -80% → 污染 -80% |
| 电炉 | 能效 2 ×2 | 电耗 -80% → 源头污染 -80% |
| 组装机 3 | 能效 3 ×4 | 电耗 -80% |
| 化工厂 | 能效 3 ×3 | 电耗 -80% |
| 炼油厂 | 能效 3 ×3 | 电耗 -80% |

**能效模块是你降污染的最强工具。** 产能模块增加污染（速度 + 功耗 ↑↑），不要为了生产力插件牺牲污染。

---

## 第五刀：加速科研（砍进化因子根源）

污染驱动虫子进化。加快科研 = 在虫子进化到下一级之前通关 = 间接降低污染的影响：

```
进化因子增加来源：
  时间流逝（缓慢）
  污染覆盖虫巢（主要） ← 你的污染越低，进化越慢
  虫巢被摧毁（一次性小幅增加）
```

---

## 第六刀：主动清除虫巢

按 `P` 打开生产统计 → 污染标签 → 看地图上哪些虫巢在吃你的污染云。用坦克/蜘蛛机甲过去把它们推掉：

```
清除虫巢 = 切断进化因子来源 + 停止被污染刺激的进攻
```

但不要杀过头——杀巢本身会增加进化因子（虽然幅度小）。

---

## 不该做的事

| 错误做法 | 为什么错 |
|------|------|
| ❌ 满工厂铺能效信标 | 信标本身巨耗电，抵消了能效收益 |
| ❌ 停产降污染 | **工厂不能停！** 应该用插件优化而非减产 |
| ❌ 只用太阳能但不切 | 太阳能白天产、锅炉晚上补 → 锅炉仍然在烧 |
| ❌ 种树 | 树吸污染效率极低，不是对策 |

---

## 执行清单（按优先级）

```
第 0 步：矿机全上能效 1 ×3（解锁能效模块后立刻做）
第 1 步：锅炉切太阳能 + 蓄电池（白天多余电存入，晚上少烧煤）
第 2 步：石炉/钢炉换电炉 + 能效 2
第 3 步：所有组装机/化工厂/炼油厂上能效模块
第 4 步：推掉污染云覆盖的虫巢
第 5 步：切核电（彻底告别煤炭发电）
```

---

## 各阶段效果汇总

| 阶段 | 污染削减 | 措施 |
|------|------|------|
| 没做 | 100% 基线 | — |
| 矿机能效 | -80% 矿机污染 ≈ **-32% 总污染** | 能效 1 ×3 |
| 清洁电力 | -10% 总污染 | 太阳能 + 蓄电池 |
| 电炉能效 | -25% 总污染 | 电炉 + 能效 2 ×2 |
| 全厂能效 | -10% 总污染 | 组装机 + 化工厂 |
| 清除虫巢 | 虫子不再吃你的云 | 主动清剿 |
| **合计** | **↓ ~75% 总污染** | — |

---

## 一句话

> 矿机三颗绿1能效 → 电炉两颗绿2能效 → 太阳能 + 蓄电池 → 推掉吃云虫巢。四步走完，污染云缩到原来的四分之一，虫子基本不来了。
