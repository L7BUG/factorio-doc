# 激光塔 + 石墙防线设计

> 适用版本：Factorio 2.0 + Space Age DLC

---

## 核心参数速查

| 参数 | 值 | 说明 |
|------|------|------|
| 激光塔-墙间距 | **1 格** | 防止沙虫溅射 |
| 主墙厚度 | **2~3 层** | 2 层日常，3 层后期 |
| 激光塔间隔 | **2 格** | 火力重叠 |
| 龙牙距主墙 | **2 格** | 缓冲溅射 |
| 龙牙形态 | 单格散列，间隔 1 格 | 迷宫效果 |

---

## 间距为什么是 1 格

| 间距 | 效果 |
|------|------|
| **0 格（贴脸）** | 虫子啃墙时，大型沙虫的范围溅射会波及激光塔 |
| **1 格** ✅ | 溅射打不到塔，塔的射程覆盖墙的每一格 |
| **2 格+** | 无额外好处，墙离塔越远，塔覆盖墙的范围越小 |

> 激光塔射程 24 格，绿虫（Behemoth Spitter）射程 16 格。虫子在墙外吐痰打不到 1 格后的激光塔。虫子走近啃墙时，塔在射程内。

---

## 墙厚：按进化阶段

| 进化阶段 | 推荐厚度 | 原因 |
|----------|----------|------|
| 前期（小虫/中虫） | **单层** | 够用，省石头 |
| 中期（大虫） | **2 层** | 大虫啃单层太快 |
| 后期（巨型虫/绿虫） | **3 层 + 龙牙** | 给激光塔争取击杀时间 |

## 龙牙（Dragon's Teeth）

### 原理

```
   █  █  █  █  █  █  █  █      ← 龙牙（单格散列，间距 1）
   ░  ░  ░  ░  ░  ░  ░  ░
   ████████████████████████████  ← 主墙（2-3 层）
   ░░░░░░░░░░░░░░░░░░░░░░░░░░   ← 1 格空隙
   🔫  🔫  🔫  🔫  🔫  🔫  🔫    ← 激光塔

虫子 → 撞上龙牙 → 绕路 → 再撞 → 再绕 → 塔持续输出
原来 2 秒到墙 → 现在 8 秒 → 多挨 6 秒激光
```

龙牙不挡子弹也不防溅射——它的唯一作用是**扰乱虫子的寻路**。虫子 AI 会尝试绕过障碍而非直接啃，单格散列让它反复转向。

### 放置方式

```
1. 在主墙外 2 格处开始放
2. 用单格石墙，间隔 1 格
3. 一排就够（不需要多层龙牙，多层反而挡不住自己的机器人维修路径）
```

## 激光塔间距：为什么是 2 格

```
间隔太大：
  🔫              🔫              🔫
  塔间距 > 3 → 中间火力稀薄 → 虫子集中攻弱处

间隔 2 格 ✅：
  🔫  🔫  🔫  🔫  🔫  🔫
  射程 24 格，射程圈大量重叠
  任何角度来的虫子至少被 2-3 座塔同时集火
```

## 蓝图建造顺序

```
1. 铺激光塔（间隔 2 格），用电力杆供电
2. 塔外 1 格处铺主墙 2-3 层
3. 墙外 2 格处放龙牙：单格墙，间隔 1 格
4. 用蓝图保存 → 拖拽铺开
```

## 注意事项

| 注意 | 说明 |
|------|------|
| ⚡ **电力** | 激光塔打起来巨耗电，确保电力充足。必要时加蓄电池缓冲 |
| 🤖 **维修** | 后期配合建筑机器人 + 维修包自动修墙，防线可以永久不坏 |
| 🚧 **拐角** | 拐角处虫子攻击面大，加厚 1-2 层墙，多放 1-2 座激光塔 |
| 🚪 **出入口** | 留门！用普通传送带门或留 2 格宽的口供你自己出入 |
| 🔥 **喷火塔** | 后期混合激光 + 喷火塔效果更佳——激光打漏掉的，喷火清大片 |
