# 异星工厂核电完全指南

> 适用：Factorio 2.0 + Space Age DLC

---

## 核电优势

| | 锅炉烧煤 | 太阳能 | **核电** |
|------|------|------|------|
| 功率密度 | 低 | 极低 | **极高**（480 MW 只需 4 个反应堆） |
| 污染 | 高 | 零 | **零** |
| 燃料消耗 | 持续烧煤 | 无 | 1 根铀棒烧 200 秒 |
| 占地 | 中 | 极大 | **极小** |
| UPS | 中 | 零 | 低 |

---

## 第一步：采铀矿

```
找到铀矿脉（地图上深绿色）→ 电矿机开采 → 铀矿石

需要硫酸：矿机有独立的管道接口，接硫酸过去
```

---

## 第二步：铀处理

```
铀矿石 → 离心机（铀处理）→ U-238（99.3%）+ U-235（0.7%）

一个离心机处理 10 矿石/12 秒
每 2000 秒 约出 1 个 U-235 ≈ 每 33 分钟出一个
```

> U-235 极稀有，但别慌——1 个 U-235 = 10 根燃料棒，够 1 个反应堆烧 2000 秒。

---

## 第三步：Kovarex 富集（U-235 无限增殖）

有了第一个 U-235 后，别立刻做成燃料棒——先解锁 Kovarex：

```
40 U-235 + 5 U-238 → Kovarex 离心机 → 41 U-235 + 2 U-238（60 秒）

本质：5 U-238 → 1 U-235 的转换
把 40 个 U-235 锁在离心机里做"种子" → 它用 U-238 不断增殖
U-235 越生越多 → 永无止境
```

**攒够 40 个 U-235 启动 Kovarex → 从此铀燃料永不枯竭。**

---

## 第四步：造燃料棒

```
1 U-235 + 19 U-238 + 10 铁板 → 组装机 → 10 根铀燃料棒
```

1 根燃料棒在单个反应堆里烧 200 秒（≈ 3.3 分钟）。

---

## 反应堆核心机制：邻居加成

**反应堆相邻排列时，每个相邻面互相加成 100% 热量：**

```
1 个孤堆：          40 MW

2 个相邻：          40 × 2 × 2 = 160 MW（每个 80 MW）

4 个 2×2 方阵：     40 × 4 × 3 = 480 MW（每个 120 MW）
                   每个堆有两个相邻面 = +200%
                   4 个堆各 120 MW = 480 MW 总功率
```

| 布局 | 反应堆 | 总功率 | 每堆功率 |
|------|------|------|------|
| 1×1 | 1 | 40 MW | 40 MW |
| 2×1 | 2 | 160 MW | 80 MW |
| **2×2** | **4** | **480 MW** | **120 MW** |
| 2×3 | 6 | 800 MW | 133 MW |
| 2×4 | 8 | 1120 MW | 140 MW |
| 2×N 极限 | 2N | — | 逼近 160 MW/堆 |

## 热量 → 蒸汽 → 电力：配比表

```
反应堆  →  热管  →  热交换器  →  蒸汽  →  汽轮机  →  电力
(热源)             (10 MW热/台)  (500°C)  (60蒸汽/s)
```

### 2×2 四堆（480 MW）标准配比

| 组件 | 数量 | 说明 |
|------|------|------|
| 核反应堆 | **4** | 2×2 排列 |
| 热交换器 | **48** | 480 MW ÷ 10 MW/台 |
| 汽轮机 | **~83** | 理论 82.5，多放 1-2 个 |
| 水泵 | **2~4** | 1 个泵供 11-12 个热交换器 |
| 热管 | 尽量短 | 长距离热管衰减严重（>10 格开始掉温度） |

### 通用公式（2×N 布局）

```
热交换器 = 32N − 16
汽轮机   = 55N − 27
水泵     = ~0.28N（向上取整）
总功率   = 320N − 160 MW
```

## 2.0 重大改进：反应堆可以直接接线

老版本需要复杂的**蒸汽储罐 + 电路**来控制燃料棒投放。2.0 大幅简化：

```
反应堆 → 红/绿线 → 可直接读取：
  ☑ 温度
  ☑ 燃料状态
  ☑ 热量输出

控制方式：
  条件：温度 < 550°C → 机械臂投放 1 根燃料棒
  结果：反应堆自动控温，不浪费燃料
```

> 不再需要蒸汽罐来检测「是否需要加燃料」。2.0 直接读温度就行。

## 防浪费技巧

### 铀燃料棒不会烧完自动停——不烧也消耗

```
反应堆只要插了燃料 → 200 秒倒计时 → 不管实际产热多少，一定烧完
→ 热量产出来没人用 → 浪费
```

### 2.0 最优控法

```
反应堆 → 电路 → 判断器：
  温度 < 550°C → 机械臂允许插入 1 根 → 机械臂用「栈大小 = 1」
  
不需要蒸汽罐、不需要复杂逻辑。3 个线 + 1 个判断器搞定。
```

## 从零到核电的步骤

```
1. 找到铀矿 → 用火车或传送带把铀矿运回基地
2. 离心机处理铀矿 → 攒 U-235（耐心，要 33 分钟出第一个）
3. 第一个 U-235 → 解锁 Kovarex 富集科技
4. 继续攒到 40 个 U-235 → 启动 Kovarex 循环
5. Kovarex 开始增殖 → U-235 不再缺
6. 造铀燃料棒 → 建 2×2 反应堆 → 扔燃料进去
7. 接热管 → 热交换器 → 汽轮机 → 电网
8. 接电路控制温度 → 永动运行
```

## 注意事项

| 问题 | 解决 |
|------|------|
| 🔥 热管太长 | 热管超过 10-15 格后热量传不过去，温度掉到 500°C 以下热交换器不工作 |
| 💧 水泵不够 | 1 个泵 = 1200 水/秒 ≈ 供 11-12 个热交换器。2×2 核电站至少 2-3 个泵 |
| 💀 反应堆被毁 | 反应堆被虫子啃了会核爆（小范围高伤害）——反应堆周围放墙 + 炮塔 |
| ⚠️ 别手贱 | 采矿铀矿后手上有铀矿可以随时扔——没有辐射危险，不会毒你 |
