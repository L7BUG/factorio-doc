## 基础概念

### 什么是均分器

均分器用分流器（Splitter）+ 地下传送带 + 红绿带组合，将多条传送带的物品**均匀分配**到多条输出带。

### 三条核心规则

```
1. 输入每条带上的物品 → 进入均分器的分流器网络
2. 分流器不断交叉混合 → 每条输出带包含所有输入带的物品
3. 最终结果：每条输出带 = 总输入流量 ÷ 输出带数
```

### 合集命名规则

```
命名格式： N→M [类型] [子类型]

  6→2              N=6输入, M=2输出
  TU               吞吐量无限制
  Lane             通道均衡器
  Corner           转角版
  Split            分流版
  Merge            合并版
  Yellow / Red     黄带/红带
```

## 1. 普通 Balancer

### 是什么

最基础的均分器。把 N 条输入带的物品**均分到 M 条输出带**上。每条输出带包含来自所有输入带的等比例物品。

### 工作原理

```
3→2 Balancer 内部：

  输1 ──S──┬──S── 出1
            │
  输2 ──S──┤
            │
  输3 ──S──┴──S── 出2

  S = 分流器，每个 S 把输入对半劈
  经过两层交叉 → 出1 和 出2 各含输1:输2:输3 = 1:1:1
```

### 场景

```
矿井 → 6 条带 → 6→2 Balancer → 2 条满带去工厂
```

不管 6 条带各自流量不同（有些矿多有些矿少），出来 2 条带一定均匀。

### 局限

```
部分输入口被堵 → 内部某些分流器路径断连 → 剩余口的总吞吐量打折

6→2，堵了 3 个输入 → 按说剩余 3 个输入应该能跑满 2 条输出
                     但普通 Balancer 做不到 → 可能只出 1.3 条的量
```

### 合集里有多少

| 范围 | 示例 |
|------|------|
| 2→x | 2-4, 2-5, 2-6, 2-7, 2-8, 2-9 |
| 3→x | 3-5, 3-6, 3-7, 3-8, 3-9 |
| 4→x | 4-2, 4-5, 4-6, 4-7, 4-8, 4-9 |
| 5→x ~ 9→x | 全覆盖 |
| 巨型 | 10-10, 12-12, 16-16, 32-32, 64-64, 128-128 |

共 **79 个**。

## 2. TU Balancer

### 是什么

**Throughput Unlimited（吞吐量无限制）**。普通 Balancer 的增强版。**任意几个输入口或输出口被堵，剩余口的流量不折损，满带宽运行。**

### 工作原理

普通 Balancer 的内部分流器拓扑在某些口被堵时会产生瓶颈。TU Balancer **额外加了交叉层级**，确保任意输入都能通过多条路径抵达任意输出。

```
普通 3→2：                3→2 TU：

  输1 ─S─┬─S── 出1         输1 ─S─┬─S─┬─S── 出1
          │                       │   │
  输2 ─S─┤                 输2 ─S─┼─S─┤
          │                       │   │
  输3 ─S─┴─S── 出2         输3 ─S─┴─S─┴─S── 出2

  堵了输3：                 堵了输3：
  出1 + 出2 < 输1+输2     出1 + 出2 = 输1+输2 ✅ 满带宽
```

### 场景

```
卸货站 → 某几个箱子满了 → 对应机械臂停

普通 Balancer： 停了的几条带拖累整个均分器，吞吐打折
TU Balancer：   自动绕路，还在卸的箱子满速运行
```

### TU 是普通 Balancer 的纯上位替代

| | 普通 | TU |
|------|------|------|
| 全接满时 | ✅ 均分 | ✅ 均分 |
| 部分口堵了 | ❌ 吞吐打折 | ✅ 满带宽 |
| 占地 | 较小 | 较大（多几层分流器）|
| **能替代对方吗** | ❌ | ✅ |

> TU 包含了普通 Balancer 的全部功能。唯一的代价是占地稍大。

### 合集里有多少

| 范围 | 示例 |
|------|------|
| 1→x | 1-3 TU ~ 1-9 TU |
| 2→x | 2-3 TU, 2-4 TU, 2-8 TU |
| 3→x | 3-1 TU, 3-2 TU, 3-3 TU, 3-4 TU, 3-6 TU |
| 4→x | 4-1 TU ~ 4-4 TU, 4-6 TU, 4-8 TU, 4-12 TU |
| 5→x ~ 9→x | 5-1 TU ~ 9-1 TU |
| 8→x | 8-1 TU, 8-2 TU, 8-4 TU, 8-8 TU |
| 其他 | 12-4 TU, 16-8 TU |

共 **32 个**。

## 3. Lane Balancer

### 是什么

**通道均衡器**。解决的是**单条传送带内部左右两路**的不均衡。

### 传送带的两路结构

每条传送带有左路和右路，各自独立运货：

```
████|████   ← 左右均衡
████|░░░░   ← 只有左路有货（机械臂单侧投放的典型效果）
░░░░|████   ← 只有右路有货
```

普通 Balancer **不管左右路**——它只在带与带之间做均分。带内左右失衡原封不动穿过去。

### 工作原理

```
1-1 Lane Balancer（2×2 格）：

  输: ██|░░
        │
    ┌───┴───┐
    │ 地下带  │  ← 拆分左右路，交叉重组
    └───┬───┘
        │
  出: █░|█░   ← 左右均匀分布
```

### 场景

你的卸货站机械臂全在同一侧：

```
24 条带全 ██|░░ → 先过 Lane Balancer → 24 条全 ██|██ → 再过普通 Balancer 收窄

没有 LB → 后面的 6→2、8→4 怎么均分都没用——左右路永远偏的。
```

### LB 和普通 Balancer 的区别

| | 普通 Balancer | Lane Balancer |
|------|------|------|
| 管什么 | 带与带之间 | 带内左路 vs 右路 |
| N→M 不等宽 | ✅ | ❌ 只有等宽（N→N）|
| 左右路均衡 | ❌ | ✅ |
| 2→2 LB 有带间均衡吗 | — | ✅（等宽时自带）|

> **LB 和 Balancer 是正交的**——解决完全不同维度的问题。LB 只有等宽版本（1-1, 2-2, 3-3, 4-4, 6-6, 8-8），没有 6→2 LB 这种东西。

### 合集里有多少

| 蓝图 | 尺寸 |
|------|------|
| 1-1 TU lane balancer | 2×2 |
| 1-1 TU lane balancer (alt) | 2×2（变体） |
| 2-2 lane balancer | 2×3 |
| 3-3 lane balancer | — |
| 4-4 lane balancer | — |
| 4-4 lane balancer (alt) | — |
| 6-6 lane balancer | — |
| 8-8 lane balancer | — |

共 **8 个**。

## 4. Corner Balancer

### 是什么

**转角均分器**。普通均分器的输入和输出在同一直线上（两端进出）。Corner 版本把**输出方向转了 90°**。

```
普通：                       Corner：

  →→→ [均分器] →→→           →→→ [均分器]
                                                   ↓
                                                   ↓
                                                   ↓
```

### 什么时候用

工厂布局受限，输出必须拐弯走。功能和普通/TU/LB 相同，只是物理形状不同。

### 关键：不是所有输入输出口都在转角

有些 Corner 是**输入-输出在相邻边**，有些是**内部路由拐弯但对外还是一字排开**。看预览图确认。

### 合集里有多少

| 蓝图 | 类型 |
|------|------|
| 1-1 TU lane corner | Lane + TU |
| 2-2 lane corner | Lane |
| 3-3 TU corner | TU |
| 4-4 lane corner | Lane |
| 4-4 TU corner | TU |
| 6-6 corner | 普通 |
| 8-8 corner | 普通 |

共 **7 个**。

## 5. Split Balancer

### 是什么

**分流均分器**。输入一条带（或多条），输出时**按指定比例分配到不同的输出带**——不是均匀分配，而是"这条路分给 A 区、那条路分给 B 区"。

```
普通 4→4：每条出带各 25%
Split：    出1 = 30%, 出2 = 20%, 出3 = 50%, 出4 = 0% (可配置)
```

### 场景

```
铁板主传送带 → 绿电路区优先 60%、红电路区 30%、其他 10%
```

Split 保证各区拿到的比例固定，不受上游波动影响。比用分流器+优先级链更精确。

### 合集里有多少

- 4-4 TU split balancer
- 8-8 split balancer

共 **2 个**。

---

## 6. Merge Balancer

### 什么

**合并均分器。** Split 的反向——多条带按比例合流到更少的输出带上，同时保持均衡。

### 场景

```
4 个矿区的 4 条带,矿区产出量不同 → Merge 合并 → 2 条满带宽带
```

### 合集里有多少

- 4-4 TU merge balancer
- 8-8 merge balancer

共 **2 个**。

## 7. 降级版

### 是什么

普通/TU Balancer 默认使用蓝带（45件/秒）。降级版用**黄带（15件/秒）或红带（30件/秒）**设计，省资源。

### 什么时候用

早期游戏没有蓝带，但需要均分器。或者你明确只需要低速均分（比如红瓶、绿瓶产线根本跑不满蓝带）。

### 合集里有多少

**黄带（Yellow）**：21 个，覆盖 4→x 到 9→x  
**红带（Red）**：1 个（16-16）

共 **22 个**。

## 类型对比总结

| | 普通 | TU | Lane | Corner | Split | Merge |
|------|------|------|------|------|------|------|
| 带间流量均分 | ✅ | ✅ | 等宽时附带 | 取决于基底 | ❌ | ✅ |
| 部分口堵了仍满带宽 | ❌ | ✅ | 取决于基底 | 取决于基底 | — | — |
| 带内左右路均衡 | ❌ | ❌ | ✅ | 取决于基底 | ❌ | ❌ |
| N→M 不等宽 | ✅ | ✅ | ❌ | ❌/✅ | ❌ | ✅ |
| 输出方向 | 直线 | 直线 | 直线 | **拐弯** | 直线 | 直线 |
| 分配方式 | 均分 | 均分 | 均分 | 均分 | **比例** | 合流 |
| 占地 | 小 | 中 | 极小 | 小 | 中 | 中 |
| 蓝图数量 | 79 | 32 | 8 | 7 | 2 | 2 |

## 选择决策树

```
你需要什么？

├── 带子内部左右路不均（████|░░░░）
│   └── ▶ Lane Balancer（1-1, 2-2, ..., 6-6, 8-8）
│
├── N 条带收窄到 M 条带（6→2, 8→4...）
│   ├── 输入口永远不会被堵（全在卸货，箱子一直在消耗）
│   │   └── ▶ 普通 Balancer
│   └── 输入口可能被堵（箱子满了，机械臂停）
│       └── ▶ TU Balancer
│
├── 既需要左右路均衡 + 又需要带间收窄
│   └── ▶ Lane Balancer（左右路） → Balancer/TU（收窄）
│        两步分开做，不能用一个均分器同时搞定
│
├── 输出必须拐弯
│   └── ▶ Corner 版本（Lane Corner / TU Corner）
│
├── 按比例分配（A工厂60% B工厂40%）
│   └── ▶ Split Balancer
│
├── 多条带合流
│   └── ▶ Merge Balancer
│
└── 早期游戏，没蓝带
    └── ▶ Yellow / Red 降级版
```

## 常见组合套路

| 场景 | 组合 |
|------|------|
| **火车卸货站**（机械臂单侧投放） | 6-6 LB → 6-2 B → 8-2 TU |
| **主传送带均衡** | 4-4 TU（输入端直接用 TU） |
| **矿井→熔炉** | N-1 TU（矿井常枯竭，TU 抗堵） |
| **大型总线分包** | 4-4 Split → 各区按比例取货 |
| **多矿区合并** | 8-8 Merge → 4-2 TU → 统一输出 |
| **蓝带不足时期** | Yellow 降级版全套 |

## 一条实用原则

> **不确定就用 TU。** 普通 Balancer 能做的 TU 全都能做。占地多出来的几格在大多数工厂里无关紧要，而 TU 的抗堵能力总有一天会救你一命。
